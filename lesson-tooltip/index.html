Sticky tooltip

throttle(updateTooltip, 500) для котроля

Main goal:

  • Tooltip should show current mouse position (x:, y:)
  • tooltips arrow should be placed at current mouse position;
  • if mouse is not moving tooltip should not be updated

Stretch goal:

  • prevent tooltip from updating too often
    • 1st time update immediately on move
    • next times update only after a given delay (500ms)
    • the last time after the mouse stopped
  • implement delay behaviour as a decorator (throttle)
